# Live Object Detection & Tracking App

## 📷 Description
This project is a real-time object detection web app using Streamlit and YOLOv8.

## 🚀 Features
- Live webcam detection
- Object tracking
- Object counting
- Bounding boxes and labels

## 🧠 Technologies Used
- Python
- Streamlit
- YOLOv8 (Ultralytics)
- OpenCV

## ▶️ How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
